### 142

```
오늘의 메뉴: 김밥

오늘의 메뉴: 튀김
```
```python
리스트 = ["김밥", "라면", "튀김"]
for 음식 in 리스트:
    print("오늘의 메뉴:", 음식) 
```

### 144
리스트에는 동물이름이 문자열로 저장돼 있다.
```
리스트 = ['dog', 'cat', 'parrot']
```
동물 이름과 글자수를 다음과 같이 출력하라.
```
dog 3
cat 3
parrot 6
```
```python
리스트 = ['dog', 'cat', 'parrot']
for 동물 in 리스트:
    print(동물, len(동물))
```

### 146
리스트에는 세 개의 숫자가 바인딩돼 있다.
```
리스트 = [1, 2, 3]
```

```

3 x 2
3 x 3
```
```python

for 변수 in 리스트:

```
