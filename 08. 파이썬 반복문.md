## 08. 파이썬 반복문

### 131
for문의 실행결과를 예측하라.
```
과일 = ["사과", "귤", "수박"]
for 변수 in 과일:
    print(변수)
```    
```python
사과
귤
수박
```

### 134
for문을 풀어서 동일한 동작을하는 코드를 작성하라.
```
for 변수 in ["A", "B", "C"]:
  print("출력:", 변수)
```
```python
출력: A
출력: B
출력: C
```
