## 003 print 기초

화면에 아래 문장을 출력하세요. (중간에 "가 있음에 주의하세요.)

## 005 print 탭과 줄바꿈

다음 코드를 실행해보고 \t와 \n의 역할을 설명해보세요.

```
print("안녕하세요.\n만나서\t\t반갑습니다.")
```

```python
print("안녕하세요. \n만나서\t\t반갑습니다.")
# \n은 줄바꿈, \t는 탭
```



## 006 print 여러 데이터 출력

print 함수에 두 개의 단어를 입력한 예제입니다. 아래 코드의 출력 결과를 예상해봅시다.

```
print ("오늘은", "일요일")
```

```python
오늘은 일요일
```
