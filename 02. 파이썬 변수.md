## 012 변수 사용하기
다음 표는 삼성전자의 일부 투자정보입니다. 변수를 사용해서 시가총액, 현재가, PER 등을 바인딩해보세요.

항목	값
시가총액	298조
현재가	50,000원
PER	15.79
```python
시가총액 = 298000000000000
현재가 = 5000

print(시가총액, type(시가총액))
```

## 013 문자열 출력
변수 s와 t에는 각각 문자열이 바인딩 되어있습니다.
```
>> s = "hello"
>> t = "python"
```
두 변수를 이용하여 아래와 같이 출력해보세요.
```
실행 예:
hello! python
```
```python
s = "hello"
t = "python"
print(s+"!",t)
```
