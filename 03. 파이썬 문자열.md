# 03. 파이썬 문자열

## 021 문자열 인덱싱
letters가 바인딩하는 문자열에서 첫번째와 세번째 문자를 출력하세요.
```
>> letters = 'python'
```
```
실행 예
p t
```
```python
letters='python'
print(letters[0], letters[2])
```

## 022 문자열 슬라이싱
자동차 번호가 다음과 같을 때 뒤에 4자리만 출력하세요.
```
>> license_plate = "24가 2210"
```
```
실행 예: 2210
```
```python
license_plate="24가 2210"
print(license_plate[-4:])
```
